# Atlas Journey — MVP funcional

Primera versión de prueba de una plataforma para registrar ciudades visitadas, guardar diarios y fotos, y compartir consejos con otros viajeros.

## Funciones incluidas

- Registro e inicio de sesión local.
- Globo 3D interactivo.
- Ciudades visitadas y viajes planificados.
- Diario, consejos, valoración y fotografías.
- Experiencias públicas o privadas.
- Sección Explorar.
- Edición del perfil, viajes y estadísticas.

## Cómo probarla

1. Descomprime el ZIP.
2. Abre `index.html` con Chrome, Safari o Edge.
3. Crea una cuenta de prueba.

El globo 3D y la búsqueda online necesitan conexión a internet. Los datos se guardan únicamente en el navegador durante esta primera fase.
